# Stories: Dream Team Dashboard

## Story 1: Структура и навигация
**Описание:** HTML/CSS каркас дашборда: header с логотипом Dream Team, боковая панель навигации, основная область контента.
**Acceptance criteria:**
- [ ] Header с названием "Dream Team Dashboard"
- [ ] Sidebar: Projects, Agents, Activity
- [ ] Responsive (desktop + tablet)
- [ ] Тёмная тема (dark navy + акценты)
**Агент:** developer-frontend
**Приоритет:** P1
**Оценка:** 1.5 часа

## Story 2: Карточки проектов
**Описание:** Список всех проектов команды с карточками: название, статус, последний коммит, прогресс пайплайна, ссылка на GitHub.
**Acceptance criteria:**
- [ ] Карточка проекта: имя, описание, статус (active/completed), GitHub link
- [ ] Прогресс-бар пайплайна (artifacts → plan → implement → test → review → deploy)
- [ ] Последний коммит (сообщение, дата, автор)
- [ ] Статистика: кол-во файлов, строк кода
- [ ] Данные из конфигурационного JSON (projects.json)
**Агент:** developer-frontend
**Приоритет:** P1
**Оценка:** 2 часа

## Story 3: Панель агентов
**Описание:** Список агентов команды с ролями, моделями, текущими задачами.
**Acceptance criteria:**
- [ ] Карточка агента: id, роль, модель, trust level
- [ ] Статус: idle / working / error
- [ ] Текущая задача (если есть)
- [ ] Данные из agents.json
**Агент:** developer-frontend
**Приоритет:** P1
**Оценка:** 1.5 часа

## Story 4: Лента активности
**Описание:** Хронологическая лента событий: коммиты, завершение пайплайнов, QA отчёты, деплои.
**Acceptance criteria:**
- [ ] Лента с иконками по типу события (commit, test, deploy, review)
- [ ] Временные метки
- [ ] Фильтр по проекту
- [ ] Данные из activity.json
**Агент:** developer-frontend
**Приоритет:** P2
**Оценка:** 1.5 часа

## Story 5: Статистика и метрики
**Описание:** Блок с общей статистикой: всего проектов, строк кода, коммитов, багов найдено/пофикшено.
**Acceptance criteria:**
- [ ] Метрики: total projects, total LOC, total commits, bugs found/fixed
- [ ] Анимированные счётчики при загрузке
- [ ] Визуально выделенный блок сверху дашборда
**Агент:** developer-frontend
**Приоритет:** P2
**Оценка:** 1 час

## Story 6: Дизайн и polish
**Описание:** Анимации, hover-эффекты, иконки, шрифты, финальная полировка.
**Acceptance criteria:**
- [ ] Smooth анимации при загрузке карточек
- [ ] Hover-эффекты на карточках
- [ ] Font Awesome или Material Icons
- [ ] Google Fonts (Inter)
- [ ] Все тексты на русском и английском (mixed)
**Агент:** developer-frontend
**Приоритет:** P2
**Оценка:** 1 час

## Story 7: E2E тестирование
**Описание:** Проверка всех экранов, responsive, данных, навигации.
**Acceptance criteria:**
- [ ] Все секции отображаются корректно
- [ ] Responsive на мобиле (320px+)
- [ ] Навигация работает
- [ ] Данные из JSON загружаются
- [ ] Нет console errors
**Агент:** tester
**Приоритет:** P1
**Оценка:** 1 час

---

**Итого:** 7 stories, ~10 часов
